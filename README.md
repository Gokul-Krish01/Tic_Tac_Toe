# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript. The game can be played in single-player mode against the computer or in multiplayer mode between two players.

## Features

- **Single Player Mode**: Play against the computer.
- **Multiplayer Mode**: Play with a friend.
- **Responsive Design**: The game is playable on all screen sizes.
- **Reset Game**: Start a new game at any time.
- **Winner Announcement**: Display the winner with a modal popup.
